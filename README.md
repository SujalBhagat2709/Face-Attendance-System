# Face Attendance System

## Overview
A simple AI-style attendance system using face image datasets.

---

## Features
- Load face images
- Store person labels
- Mark attendance with timestamps

---

## Files
- `load_faces.py` → loads face images from folders
- `attendance.py` → records attendance entries

---

## Dataset Structure

```text
faces/
├── person1/
├── person2/
```

---

## Usage

```bash
python load_faces.py
python attendance.py
```

---

## Example Output

```text
{'name': 'Sujal', 'time': '10:30:15'}
```
